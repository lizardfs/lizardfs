lfsdirinfo(1)
=============

== NAME

lfsdirinfo - show directories stats

== SYNOPSIS

[verse]
*lfsdirinfo* [*-n*|*-h*|*-H*] 'OBJECT'...

== DESCRIPTION

*lfsdirinfo* is extended, LizardFS-specific equivalent of *du -s* command. It prints summary for each
specified object (single file or directory tree).

== OPTIONS

*-n*, *-h*, *-H*::
These options are described in lfstools(1).

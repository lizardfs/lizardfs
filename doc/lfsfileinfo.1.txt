lfsfileinfo(1)
==============

== NAME

lfsfileinfo - locate chunks

== SYNOPSIS

[verse]
*lfsfileinfo* 'FILE'...

== DESCRIPTION

*lfsfileinfo* prints location ('chunkserver' host and port) of each chunk copy belonging to
specified file(s). It can be used on any file, included deleted ('trash').

lfshdd.cfg(5)
=============

== NAME

lfshdd.cfg - list of LizardFS storage directories for lfschunkserver

== DESCRIPTION

The file *lfshdd.cfg* contains list of directories (mountpoints) used for LizardFS storage (one per
line). Directory prefixed by *** character causes given directory to be freed by replicating all
data already stored there to another locations. Lines starting with *#* character are ignored.
